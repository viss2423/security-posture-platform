# Security Posture Platform (Student Lab)

Status: Work in progress.

A student-built platform that models an organization (users, hosts, web assets), ingests security signals, and visualizes risk and posture in Grafana.

## Safety & Scope
Authorized testing only. External web scanning is limited to verified domains and runs non-destructive exposure checks.
